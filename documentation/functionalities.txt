1. Required functionalities :
- Show a predefined sized grid in html+css
- Calculate a minefield with a predefined size and number of mines
- Associate html grid and minefield
- On left-click, discover a case
- If you left-click on a mine, you lose
- If you left click all the non-mine cases, you win
- Calculates the time taken to solve the game

2. Advanced functionalities
- Show a grid sized with parameters in html+css
- If you left click on a blank case (no mine around), show all touching blank cases
- Easy, Medium, Hard mode
- Custom mode (you enter the size and the number of mines)
- Restart button
- Share games
- Show a real time timer
- Share games with best timer...
- Share games with best timer as a decreasing counter (challenge mode)
- Keyboard shortcuts

3. Cosmetic functionalities
- On right-click, put a flag on a case if it doesn't have one, remove it if there is one
- Beautiful cases
- petite animation pour Game Over
- Effect when you click on cases
- Beautiful real time timer
